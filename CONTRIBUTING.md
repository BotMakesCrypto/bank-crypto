# Contributing to BankCrypto

Thank you for your interest in contributing to BankCrypto! This document provides guidelines for participating in the project.

## Ways to Contribute

### Hold BANKC
The simplest way to support BankCrypto is to buy and hold BANKC tokens. This increases demand and helps build the liquidity needed for major exchange listings.

- [Buy on Uniswap](https://app.uniswap.org/swap?outputCurrency=0xa593decc9100f0014ff8ee07735f7b8eefb7e055)
- [Buy on SushiSwap](https://www.sushi.com/ethereum/swap?token0=0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48&token1=0xa593decc9100f0014ff8ee07735f7b8eefb7e055)

### Spread the Word
- Share BankCrypto on social media
- Write about BANKC in crypto communities
- Create content (videos, articles, memes)

### Participate in Governance
- Join [GitHub Discussions](https://github.com/BotMakesCrypto/bank-crypto/discussions)
- Vote on proposals
- Suggest new features

### Code Contributions
- Submit bug reports via GitHub Issues
- Propose improvements via Pull Requests
- Review existing code

## Community Guidelines

- Be respectful and constructive
- No spam or self-promotion
- Focus on building value for the ecosystem
- Follow the [Code of Conduct](https://github.com/BotMakesCrypto/bank-crypto/blob/main/CODE_OF_CONDUCT.md)
