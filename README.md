# BankCrypto (BANKC)

[![Website](https://img.shields.io/badge/Website-bankcrypto.com-gold)](https://bankcrypto.com)
[![Twitter](https://img.shields.io/badge/Twitter-@BankCrypto__Com-blue)](https://x.com/BankCrypto_Com)

BankCrypto is a DeFi project bridging traditional banking with decentralized finance. Built on Ethereum, BankCrypto issues asset-backed NFTs (Bitcoin, gold, silver) and the BANKC utility token.
